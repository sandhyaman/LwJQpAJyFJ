# 学生成绩管理系统 python840

## 项目概述

学生成绩管理系统是基于Python开发的一个Web应用程序，旨在帮助学校高效管理学生、教师、课程及成绩信息。系统包含多个功能模块，以提升教学管理工作的自动化和便捷性。

## 技术栈

- **开发语言**：Python 3.6.8
- **框架**：Django
- **数据库**：MySQL 5.7/8
- **数据库工具**：Navicat
- **开发工具**：PyCharm

## 功能模块

- **首页**：展示系统概览和通知公告。
- **个人中心**：用户个人信息和操作记录。
- **教师管理**：管理教师信息，包括增删改查等操作。
- **学生管理**：管理学生信息，包括增删改查等操作。
- **公告信息管理**：发布公告和通知，管理公告信息。
- **课程类型管理**：设置和修改课程类型。
- **课程信息管理**：管理课程详情，包括课程设置和调整。
- **选课信息管理**：处理学生选课数据。
- **成绩信息管理**：录入、查询和修改学生成绩。

## 系统角色

系统主要分为以下角色：

- 学生
- 教师
- 管理员

各角色具有不同的操作权限。

## 运行环境

- Python 3.6.8
- Django 2.x
- MySQL 5.7/8
- Navicat (用于数据库管理)
- PyCharm (推荐开发环境)

## 部署步骤

1. 安装Python 3.6.8及Django框架。
2. 安装MySQL数据库，创建相应数据库。
3. 使用PyCharm或其他IDE导入项目代码。
4. 根据需求配置环境变量和数据库连接。
5. 运行项目，完成部署。

- **自动化管理**：简化成绩录入和查询流程，提高管理效率。

- **权限控制**：角色权限分离，保障系统的使用安全。
